## Repeated rubric gaps

1. Crisis override clarity: emergency need may temporarily permit taboo-breaking.
2. Status lifecycle: proposed, active, fulfilled, broken, repaired.
3. Consumer ownership matrix for repeated BANISH/CERES/PORTO overlap.
4. Place-route distinction for sacred sites versus sacred paths/processions.
5. Cost type vocabulary: material, labor, time, access, witness, memory.
6. Evidence anchor strength: seed fixture, consumer probe, or researched source.
